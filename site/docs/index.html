Capability-based routing

Submit a task — MagiC finds the best worker automatically based on capabilities, cost, and load.

Any worker, any language

Python, Go, Node, anything. Workers are just HTTP servers. Use CrewAI, LangChain, or raw code.

Cost control built-in

Budget alerts at 80%, auto-pause at 100%. Track spend per worker, per org, per task.

Multi-step workflows

DAG orchestration with parallel execution, failure handling, and human-in-the-loop gates.

🗄️

Production storage

Memory for dev, SQLite for single-instance, PostgreSQL for production with auto-migrations.

📡

Real-time streaming

Workers stream output token-by-token via SSE. Webhooks deliver events at-least-once.
