3 Communication Patterns MOI

Pattern 1: Worker ↔ Org (mac dinh) — Tat ca traffic qua Org. Org kiem soat, log, cost track.

Pattern 2: Worker ↔ Worker (direct) — P2P cho real-time (streaming, co-edit). Org monitor nhung khong route.

Pattern 3: Worker ↔ Environment — Worker dung MCP goi tools (DB, API, files). Org co the gate access qua policy.

MCP + MCP² = BO SUNG: MCP ket noi worker ↔ tools. MCP² ket noi org ↔ workers. 2 tang khac nhau.

Data Flow & Sequences

Simple Task

User --> Gateway --> Router --> Registry (find worker) | v Worker (task.assign) | task.accept task.progress (50%) task.complete | v Evaluator (quality check) CostCtrl (record cost) Monitor (log event) | v User <-- 200 { result } <-- Gateway

Complex Workflow — DAG Parallel Execution DA CAP NHAT

Vi du: "Product Launch Campaign" — 5 steps, 2 nhanh song song:

research (step 1) / \ v v content leads <-- 2 nhanh SONG SONG (step 2) (step 4) | | v | seo | (step 3) | \ / v v [CEO APPROVAL GATE] | v outreach (step 5) <-- CHO CA 2 nhanh xong

Orchestrator xu ly tung buoc

1. Parse workflow --> build DAG 2. Tim steps khong co dependency --> step 1 (research) Router tim worker --> assign 3. Step 1 COMPLETE --> check dependents: - Step 2 (content): depends [step1 OK] --> ASSIGN - Step 4 (leads): depends [step1 OK] --> ASSIGN --> 2 tasks chay SONG SONG (goroutines) 4. Step 2 COMPLETE --> check: - Step 3 (seo): depends [step2 OK] --> ASSIGN - Step 5 (outreach): depends [step2 OK, step4 ?] --> step4 CHUA xong --> DOI 5. Step 4 COMPLETE --> check: - Step 5: depends [step2 OK, step4 OK] --> ALL MET --> approval_required: true --> DUNG LAI --> Gui notification cho CEO review 6. CEO APPROVE --> Orchestrator ASSIGN step 5 7. Step 5 COMPLETE --> Workflow DONE

Error Handling

on_failureHanh dong
"retry"Thu lai max_retries lan voi backoff
"skip"Bo qua step, tiep tuc downstream
"abort"Dung toan bo workflow, bao user
"reassign"Router chon worker KHAC, thu lai

Vi du: Step 2 (content) fail → retry 2 lan → van fail → reassign worker khac → van fail → abort. Steps 3, 5 bi BLOCK. User duoc thong bao: "Content failed. Steps 3, 5 blocked. Action required."

Worker A (content_writing) dang viet bai | | Can data thong ke --> gui worker.delegate | { capability: "data_analysis", input: {...} } v Org nhan delegate request --> Router tim Worker B (data_analysis) --> task.assign --> Worker B --> Worker B complete --> tra data ve Worker A --> Worker A tiep tuc viet voi data --> task.complete --> bai viet hoan chinh
