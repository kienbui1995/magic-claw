Hang muc 04

MagiC Protocol (MCP²)

Chuan hoa giao tiep Organization ↔ AI Workers. 5 nguyen tac (da them Human-in-the-loop va Fail-safe).

Nguyen tacGiai thich
Transport-agnostic HTTP, WebSocket, gRPC deu duoc
Capability-based Worker tu khai bao kha nang, Org giao viec theo capability
Contract-driven Moi task co contract: input, output, quality criteria
Human-in-the-loop Nguoi dung luon la "boss", approve/reject/redirect bat ky luc nao
Fail-safe Worker fail → retry/delegate/bao nguoi dung. Khong mat viec

Hang muc 08 MOI

Tech Stack

Go core + Python SDK (theo feedback ve Go performance).

Tieu chiPythonGo
PerformanceChamNhanh 10-100x
ConcurrencyAsyncIO (ok)Goroutines (tuyet voi)
AI ecosystemRat giauNgheo

De xuat: Go core + Python SDK

Framework core (Gateway, Router, Registry...) viet bang Go — nhanh, nhe, concurrent. Workers va AI logic dung Python SDK — noi co LangChain, LiteLLM, ecosystem AI. Giong cach Kubernetes lam (Go core + Python client).

magic-claw/ +-- core/ # Go - 9 modules +-- sdk/ | +-- python/ # pip install magic-claw | +-- go/ # go get magic-claw +-- protocol/ # Spec (language-agnostic) +-- examples/ # Example workers +-- docs/ # Spec + guides (cho GitHub viral)
Can quyet dinh

Go + Python hay chi Python? Go se nhanh hon nhung can hoc them. Python de bat dau hon nhung cham.
